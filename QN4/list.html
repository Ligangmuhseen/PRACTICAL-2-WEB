HTML Lists

Unordered HTML List

An unordered list starts with the (ul) tag. Each list item starts with the (li) tag. The list items will be marked with bullets (small black circles) by default: Example
  • Coffee
  • Tea
  • Milk

Ordered Lists

An ordered list starts with the (ol) tag. Each list item starts with the (li) tag. The list items will be marked with numbers by default: Example
  1. Coffee
  2. Tea
  3. Milk
